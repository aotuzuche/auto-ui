# SafeArea 安全区域

用于修复异形屏的顶部和底部区域

## 使用示例

```jsx
import { SafeArea } from 'auto-ui'
```

### 基本使用

```jsx
<SafeArea />
```

### 顶部

```jsx
<SafeArea top />
```

或者

```jsx
<SafeArea />
```

### 底部

```jsx
<SafeArea bottom />
```

## 支持属性

| 属性   | 说明 | 类型    | 默认值 |
| ------ | ---- | ------- | ------ |
| top    | 顶部 | boolean | -      |
| bottom | 底部 | boolean | -      |
