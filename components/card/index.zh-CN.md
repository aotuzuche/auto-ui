## 支持属性

| 属性        | 说明         | 类型                                  | 默认值 |
| ----------- | ------------ | ------------------------------------- | ------ |
| direction   | 内容排列顺序 | row/column/row-reverse/column-reverse | row    |
| onClick     | 点击事件     | Function                              | -      |
| aspectRatio | 宽高比       | number                                | 1      |
| shadow      | 是否有外阴影 | Boolean                               | false  |

## Card.Figure 支持属性

| 属性    | 说明                  | 类型     | 默认值 |
| ------- | --------------------- | -------- | ------ |
| image   | 图片访问地址          | String   | -      |
| title   | 图片标题，即 alt 属性 | String   | -      |
| onClick | 点击事件              | Function | -      |
